# A simple web application to show case dog facts

* This is a learning repo for containerizing python applications. 


Learning objectives:

* Fetch data from an API
* Build data validation layer using pydantic data models
* Ingest validated data into a PostgreSQL relational database
* Run an ML algorithm for clustering or classification
* Display results into a streamlit frontend
* Use separate docker containers for backend and frontend
* User docker networking to connext backend to frontend
* Deploy to a cloud platform
